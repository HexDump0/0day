# 0day

The website for 0day (hopefully) Hack Club's first CyberSecurity YSWS program. Build security tools, find and document real vulnerabilities, or reverse engineer software and hardware to earn gear.

## Run locally

You would need Node.js and npm.

```bash
npm install
npm run dev
```

Open the local URL shown in your terminal.

## Build

```bash
npm run build
```

The prod files will be written to `dist`.

The `vid` directory contains the Remotion project used for the launch video. It has its own dependencies and scripts
